# Project Summary
BRAILLEAR is a Smart Touch-to-Speech Learning System designed to enhance Braille education for visually impaired learners. This innovative web application integrates embedded hardware with real-time audio feedback, creating an interactive and accessible learning environment. The project aims to provide an affordable educational tool that empowers users through intuitive touch-based interaction and auditory responses, ultimately improving learning outcomes.

# Project Module Description
The application consists of several functional modules:
- **Landing Page**: Overview of the project and its benefits.
- **Login System**: User authentication via email and password.
- **Dashboard**: Displays hardware connection status, demo mode, and audio playback controls.
- **Sensor Grid**: Visual representation of touch sensors triggering audio feedback.
- **Event Log**: Tracks and displays timestamped events.
- **Audio Player**: Manages audio playback and volume settings.
- **Settings Panel**: Configuration of user preferences and accessibility options.
- **Contact Page**: User inquiries and feedback.
- **Serial Panel**: Manages hardware connections and demo mode functionalities.

# File Description Inventory
- **src/App.tsx**: Main application component setting up routing and context providers.
- **src/pages/**: Contains page components like Landing, Login, Dashboard, and Contact.
- **src/components/**: Reusable UI components for the application.
- **src/lib/**: Utility libraries for audio management, serial communication, and authentication.
- **public/audio/**: Directory for audio files used in the application.
- **uploads/**: Contains project documentation and reports.
- **src/components/SerialPanel.tsx**: Manages hardware connection and demo functionalities.
- **src/lib/audioManager.ts**: Handles audio playback and management.
- **src/lib/serialManager.ts**: Manages serial communication with hardware.
- **src/components/SensorGrid.tsx**: Visual touch sensor grid for user interaction.
- **src/lib/auth.ts**: Authentication manager managing user sessions and authentication state.
- **src/components/Settings.tsx**: Configuration settings for user preferences.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express (planned for future API integration)
- **Audio Management**: Web Audio API
- **Hardware Communication**: Web Serial API
- **Build Tool**: Vite

# Usage
### Installation
```bash
# Clone the repository
git clone https://github.com/braillear/webapp.git
cd braillear-webapp

# Install dependencies
pnpm install

# Build the application
pnpm run build
```

### Running the Application
1. Open the application in a supported browser.
2. Navigate to the landing page and sign in using any email/password combination to access the dashboard.
3. Use demo mode to simulate hardware events or connect actual hardware for real-time interaction.
