# BRAILLEAR Audio Files

This directory contains phonetic audio files for the BRAILLEAR system.

## File Structure

- `A.mp3` to `Z.mp3` - Alphabet pronunciation files
- `0.mp3` to `9.mp3` - Number pronunciation files

## Audio Specifications

- Format: MP3 or WAV
- Duration: ~500ms per file
- Sample Rate: 44.1kHz recommended
- Bit Rate: 128kbps minimum
- Volume: Normalized to prevent clipping

## Demo Mode

In demo mode, the system generates synthetic audio tones for each character. 
For production use, replace these with actual recorded phonetic pronunciations.

## Adding Custom Audio

1. Record clear pronunciation of each character
2. Normalize audio levels
3. Export as MP3 or WAV format
4. Name files exactly as: A.mp3, B.mp3, etc.
5. Place in this directory

## Multilingual Support

Future versions will support:
- Regional Indian languages (Hindi, Tamil, Bengali, etc.)
- International Braille standards
- Custom voice packs
